# Kirby Roulette

[Try It](https://kirby-roulette.tekiter.net)

Simple spinner roulette game made with React Three Fiber & Jotai & Framer Motion

## Running

After clone,

```
$ corepack enable
$ pnpm install
$ pnpm dev
```
